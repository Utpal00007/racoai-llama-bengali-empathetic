# racoai-llama-bengali-empathetic


# Fine-Tuning LLaMA 3.1 on Bengali Empathetic Conversations

This project fine-tunes LLaMA 3.1-8B-Instruct using Unsloth + LoRA
for empathetic Bengali dialogue generation on limited GPU resources.

## Features
- Parameter-efficient fine-tuning (LoRA)
- Strategy pattern for training method
- Evaluation using Perplexity, BLEU, ROUGE
- Human evaluation of empathy
- Experiment logging with SQLite
